日本の絶滅危惧種を知る

Save Animals in Japan

Save Animals in Japan は日本国内の絶滅危惧種を救うために開設されました。彼らを絶滅から救うにはまず彼らの存在を知らなくてはなりません。

このサイトで日本の絶滅器具種に対する理解を深め、彼らを救う方法をみんなで考えましょう。

絶滅危惧種一覧

30の種が絶滅の危機に晒されています。

アオウミガメ

体内の脂肪が青いためにアオウミガメと呼ばれる。昔から食用とされている。

アカウミガメ

アカウミガメは信仰の対象としてもしばしば用いられる。

アホウドリ

和名は人間が接近しても地表での動きが緩怠で、捕殺が容易だったことに由来する。

アマミノクロウサギ
